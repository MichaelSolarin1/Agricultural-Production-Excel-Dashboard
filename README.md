# Agricultural-Production-Excel-Dashboard
Microsoft Excel End-to-End Data Analysis &amp; Dashboard Project

🔎 Project Overview

This project presents a comprehensive agricultural production and resource utilization analysis using Microsoft Excel.

The objective was to evaluate farming efficiency, crop performance, soil impact, irrigation effectiveness, and water consumption patterns to uncover opportunities for:

• Yield optimization
• Resource efficiency improvement
• High-performing farm identification
• Irrigation effectiveness analysis
• Strategic crop planning

Using advanced Excel functions, pivot tables, and interactive dashboard design, raw farm-level data was transformed into actionable agricultural insights.

📌 Key Business Metrics (From Dashboard)

• 🚜 Total Farms: 50
• 💧 Total Water Usage: 2,836,215 m³
• 🌾 Total Agricultural Yield: 1,353 Tons
• 📊 Average Yield per Farm: 27.06 Tons
• 🌱 Average Yield per Acre: 0.11 Tons
• 🥕 Most Productive Crop: Carrot

The dashboard provides a 360° view of agricultural performance across crop types, soil categories, irrigation methods, and farm sizes.

📊 Analysis Breakdown
1️⃣ Irrigation Efficiency Analysis

Irrigation methods evaluated:
• Manual
• Rain-fed
• Sprinkler
• Drip
• Flood

Key Insights:
• Manual irrigation shows the highest yield efficiency.
• Flood irrigation demonstrates the lowest yield output.
• Controlled irrigation systems outperform traditional water-intensive methods.

This highlights the importance of efficient water distribution systems.

2️⃣ Soil Type Performance Analysis

Yield comparison across:
• Peaty
• Clay
• Sandy
• Loamy
• Silty

Key Findings:
• Silty soil produces the highest average yield (30.89 tons).
• Peaty soil records the lowest productivity.
• Soil composition significantly impacts output variation.

This insight supports soil-based crop allocation strategies.

3️⃣ Crop Yield Analysis

Crops analyzed:
• Barley
• Carrot
• Cotton
• Maize
• Potato
• Rice
• Soybean
• Sugarcane
• Tomato
• Wheat

Key Insights:
• Tomato shows the highest total yield (201.78 tons).
• Maize records the lowest yield.
• High-demand crops demonstrate strong production clustering.

4️⃣ Farm Size Influence on Yield

Farm categories:
• Small
• Average
• Large

Findings:
• Average-sized farms generate the highest yield (30 tons).
• Large farms do not necessarily outperform mid-sized farms.
• Scale does not automatically equal efficiency.

5️⃣ Water Usage vs Crop Yield Correlation

Scatter plot analysis reveals:
• Moderate positive correlation between water usage and yield.
• Excessive water use does not proportionally increase productivity.
• Resource optimization opportunities exist.
